# Manual de supervivencia a las redes sociales

Este proyecto consiste en el desarrollo de un sitio web llamado **“Manual de supervivencia a las redes sociales”**, pensado como una guía visual, simple e interactiva para reflexionar sobre el uso cotidiano de las redes sociales.

Elegí realizar este proyecto porque las redes forman parte de la vida diaria de muchas personas y, aunque no son necesariamente algo negativo, muchas veces se usan de manera automática, excesiva o poco consciente. La idea del sitio no es proponer dejar de usarlas, sino aprender a reconocer ciertos hábitos, entender cómo funcionan y encontrar formas más saludables de vincularnos con ellas.

El público objetivo está compuesto principalmente por jóvenes y adultos de entre **20 y 35 años**, usuarios frecuentes de redes sociales, que utilizan el celular como parte de su rutina diaria y pueden sentirse identificados con situaciones como el scroll infinito, la comparación social, la pérdida de tiempo o la necesidad constante de revisar notificaciones.

El sitio está organizado en diferentes secciones:

* **Inicio:** presenta la temática general del proyecto y plantea la idea principal del manual.
* **Qué nos pasa:** explica algunas consecuencias del uso excesivo de redes, como la saturación digital, la comparación social y la falta de límites.
* **Detrás de su uso:** muestra cómo ciertos recursos de diseño, como las notificaciones, los algoritmos y el scroll infinito, influyen en la permanencia dentro de las plataformas.
* **Cómo mejorar:** propone pequeños cambios y hábitos digitales para usar las redes de una manera más consciente.
* **Contacto:** permite al usuario dejar una reflexión o mensaje relacionado con su experiencia.

A nivel visual, el proyecto utiliza una estética llamativa, moderna y cercana, con predominio del color rosa, ilustraciones digitales y una composición clara para acompañar el tono reflexivo pero accesible del contenido.
